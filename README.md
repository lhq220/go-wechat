# go-wechat

golang微信sdk，支持微信公众号，微信小程序，微信支付

微信支付回调通过自定义`UnmarshalXML`实现直接解析struct

已经实现了的功能可以详见功能模块

## Usage

```
# 支持go mod
go get github.com/zhyc9de/go-wechat
```
